# Decision Log

本文档记录 AI 在高自治模式或日常开发中做出的重要决策。尤其是 `/zno-super` 模式下，AI 可以不等待用户确认而直接选择方案，但必须在这里记录决策过程。

## 1. 记录原则

1. 只记录会影响功能、架构、性能、安全、成本、部署或维护方式的重要决策。
2. 低风险实现细节可以简要记录或不记录。
3. `/zno-super` 模式下跳过用户确认的技术选型，必须记录。
4. 决策记录不能替代测试和验证。
